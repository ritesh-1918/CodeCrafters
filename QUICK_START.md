# CodeCrafters AI - Quick Start Guide

## üöÄ Getting Started in 5 Minutes

### Step 1: Access the Application
The application is ready to run. Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Step 2: Create an Account or Login

#### Option A: Use Demo Credentials
- Email: `demo@example.com`
- Password: `demo123456`

#### Option B: Register New Account
- Click "Register here" on the login page
- Fill in your details (name, email, branch, semester)
- Create a password
- Select your branch (CSE, IT, or ECE)
- Click "Register"

### Step 3: Explore the Dashboard
After logging in, you'll see:
- Welcome greeting with your branch and semester
- Real-time statistics (challenges completed, average score, streak, total attempts)
- Featured challenges
- Your achievements

### Step 4: Browse Challenges
1. Click "Challenges" in the sidebar
2. Use filters to find challenges by:
   - **Difficulty**: Easy, Medium, Hard
   - **Branch**: CSE, IT, ECE
   - **Topic**: Arrays, Strings, Algorithms, etc.
3. Search by challenge name or description

### Step 5: Solve a Challenge
1. Click the "Start" button on any challenge
2. Choose your programming language (JavaScript, Python, Java, C++)
3. Write your solution in the code editor
4. Click "Submit" to test your code
5. Use the AI Debugging Assistant to get help

### Step 6: Check Your Progress
1. Click "Progress" in the sidebar
2. View your statistics:
   - Challenges completed
   - Average score
   - Time spent coding
   - Achievements earned
3. See difficulty breakdown
4. View skill proficiency levels
5. Check your recent submissions

### Step 7: Manage Your Profile
1. Click "Profile" in the sidebar
2. Update your information:
   - Full name
   - Bio
   - Programming languages
   - Career interests
3. Click "Save Changes"

## üêõ Troubleshooting

- Check your email and password

- Refresh the page
- Check your internet connection

- Ensure JavaScript is enabled
- Try a different programming language
- Refresh and try again

### AI assistant not responding?
- Check your internet connection
- For voice: ensure microphone permission is granted
- Try text input instead
